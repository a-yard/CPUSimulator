| 设备名 | 映射地址                  | 占用地址大小 |
| ------ | ------------------------- | ------------ |
| SRAM   | `0x0f00_0000~0x0fff_ffff` | `0x01000000` |
| Flash  | `0x3000_0000~0x3fff_ffff` | `0x10000000` |
| UART   | `0x1000_0000~0x1000_0fff` | `0x1000`     |
| CLINT  | `0x0200_0000~0x0200_ffff` | `0x10000`    |
| MROM   | `0x2000_0000~0x2000_0fff` | `0x1000`     |
